<p>Данный репозиторий содержит в себе решение всех 11 задач сайта <a href="https://kispython.ru">kispython.ru</a> по дисциплине "Программирование на языке Питон". Условие каждой задачи находится в своей определенной ветке, поэтому для его просмотра необходимо выбрать соотвествующую ветку. Название ветки совпадает с названием задачи. Например, для просмотра task10 (10-ая задача) выберите ветку task10.</p>
