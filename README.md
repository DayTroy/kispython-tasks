Данный репозиторий содержит в себе решение всех 11 задач сайта <a href="https://kispython.ru" target="_blank">kispython.ru</a> по дисциплине "Программирование на языке Питон". Каждая задача находится в своей определенной ветке, поэтому для просмотра условия и решения конкретной задачи необходимо выбрать ветку, в которой расположен код реализации конкретной задачи. Название ветки совпадает с названием задачи. Например, для просмотра task10 (10-ая задача) выберите ветку task10.
